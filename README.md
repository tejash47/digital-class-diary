# 📚 Digital Class Diary

A modern full-stack classroom management system built using **React**, **TypeScript**, **Node.js**, **Express**, and **MongoDB Atlas**. It helps teachers manage classrooms, assignments, attendance, and student records through a clean and responsive interface.

---

## 🌐 Live Demo

🔗 https://digital-class-diary.onrender.com

---

## ✨ Features

- 👨‍🏫 Teacher Dashboard
- 👨‍🎓 Student Management
- 🏫 Classroom Management
- 📝 Assignment Tracking
- 📅 Attendance Management
- 🔐 Secure Authentication
- 📱 Responsive Design
- ☁️ MongoDB Atlas Database

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/tejash47/digital-class-diary.git
```

Move into the project

```bash
cd digital-class-diary
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
APP_URL=http://localhost:5173
```

Start the development server

```bash
npm start
```

---

## 📦 Environment Variables

| Variable | Description |
|----------|-------------|
| MONGODB_URI | MongoDB Atlas Connection String |
| JWT_SECRET | JWT Secret Key |
| APP_URL | Frontend URL |
